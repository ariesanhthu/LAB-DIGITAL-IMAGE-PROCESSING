\section{Program Structure}

The program works directly with image files as input. Each image is loaded into a 
NumPy array representing pixel values in either grayscale or RGB format.

\subsection{Input}

\begin{itemize}
    \item image file (PNG, JPG, JPEG).
\end{itemize}

\subsection{Output}

\begin{itemize}
    \item saved output image results (PNG)
\end{itemize}

\subsection{Project Structure}

The implementation is organized as a single Jupyter Notebook containing all components: data preprocessing, neural network definition, training procedures, and evaluation utilities.

\subsection{Class Structure}

\subsubsection{FFNeuralNetwork Class}

The main neural network class inheriting from \texttt{nn.Module}:

\begin{table}[H]
    \centering
    \caption{FFNeuralNetwork class methods}
    \label{tab:ffnn-methods}
    \begin{tabular}{|p{4cm}|p{4cm}|p{5cm}|}
    \hline
    \textbf{Method} & \textbf{Input} & \textbf{Output} \\
    \hline
    \texttt{\_\_init\_\_(config)} & Configuration dict with input\_size, hidden\_size, output\_size & Initialized model with random weights \\
    \hline
    \texttt{relu(z)} & Pre-activation tensor $z$ & ReLU-activated tensor \\
    \hline
    \texttt{relu\_derivative(z)} & Pre-activation tensor $z$ & ReLU derivative tensor \\
    \hline
    \texttt{softmax(z)} & Logits tensor $z$ & Probability distribution tensor \\
    \hline
    \texttt{forward(X)} & Input batch $X \in \mathbb{R}^{N \times 784}$ & Output probabilities $\hat{Y} \in \mathbb{R}^{N \times 10}$ \\
    \hline
    \texttt{backward(X, y\_int, lr)} & Input batch, integer labels, learning rate & Updated model parameters \\
    \hline
    \texttt{train\_step(X, y\_int, lr)} & Input batch, integer labels, learning rate & Output probabilities \\
    \hline
    \texttt{save\_weights(model, path)} & Model instance, file path & Saved state dictionary file \\
    \hline
    \texttt{load\_weights(path, config)} & File path, configuration dict & Loaded model instance \\
    \hline
    \end{tabular}
\end{table}
